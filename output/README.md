# System Output

This directory contains all artifacts generated by the system.

## Structure

- **`charts/`**: Visualization of trades, patterns, and debug charts.
- **`results/`**: JSON results from ML runs and backtests.
- **`models/`**: Saved ML models (.pkl, .pt).
- **`trades/`**: CSV logs of executed trades.
- **`logs/`**: System logs and error tracks.
- **`phases/`**: Intermediate data products from the ML pipeline (e.g., `event_catalog.csv`, `reaction_map.csv`).
